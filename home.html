Sistem Estimasi Debit Air Sungai Berbasis Radar untuk Mendukung Mitigasi Dini Banjir

Monitoring kecepatan arus dan debit sungai berbasis radar & IoT.

Cara Kerja Sistem

Demo: Simulasi Deteksi Radar di Lab

1

Radar Sensing (uRAD)

Gelombang 24GHz dipancarkan ke permukaan air. Sinyal pantulan ditangkap untuk mengukur Doppler Shift.

2

Data Processing

Raspberry Pi memproses sinyal menggunakan algoritma FFT untuk memisahkan noise lingkungan.

3

Real-time Dashboard

Data dikirim via HTTP Request ke database Supabase dan ditampilkan secara Real-Time.
